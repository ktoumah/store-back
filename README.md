# Symfony 7.1 Project

This project is built with Symfony 7.1 and is designed to be run in a Docker environment.

## Requirements

- **Operating System:** Ubuntu 22.04
- **Docker:** Docker Desktop version 4.34.2

## Installation

To set up and run the project, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/ktoumah/store-back.git
   cd store-back
<br>

2. **Execute `make run` run command 2 times in your root project:**

   ```bash
   make run
   make run
